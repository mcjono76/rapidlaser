## Image Guidelines

- **Format**: JPG for photos, PNG for logos and graphics with transparency
- **Optimization**: Compress images for web to ensure fast loading times
- **Quality**: Use high-quality images that represent the business professionally
- **Consistency**: Maintain a consistent visual style across all images

## Fallback Images

The website is configured with fallback images from Unsplash if local images are not available. However, for the best representation of your business, please replace these with actual photos of:
- Your facility
- Your equipment in action
- Real projects you've completed
- Your team at work

## Getting Images from Original Site

To download images from https://www.rapidlaser.com.au/:
1. Visit the original website
2. Right-click on images and select "Save Image As..."
3. Save to this directory with the appropriate filename
4. Or use browser developer tools to find image URLs
